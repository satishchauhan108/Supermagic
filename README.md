# Supermagic Enterprises - E-commerce Website

A modern, responsive, mobile-first e-commerce website for Supermagic Enterprises cleaning products.

## Features

- 🛍️ Product catalog with filtering and sorting
- 📱 Fully responsive design (mobile-first)
- 🎨 Modern UI with green eco theme
- 💬 Contact modal for order inquiries (no checkout system)
- 🔍 Product search and category filtering
- ⭐ Product ratings and reviews
- 📄 Pagination for products
- 📧 Newsletter subscription

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Modern ES6+ JavaScript

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```
